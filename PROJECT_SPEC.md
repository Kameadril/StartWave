# StartWave 1.0 — Project Specification

## Концепция
StartWave — современная цифровая стартовая страница.
Цель: удобный доступ к информации, сервисам и интересам.

## Ограничения текущей версии
Не делать:
- личный кабинет;
- регистрацию;
- AI-чат;
- сложную персонализацию.

## Главная страница

Структура:

1. Header:
- логотип 🌊 StartWave;
- слоган "Твоя цифровая точка старта";
- навигация;
- часы и дата.

2. Hero:
- название StartWave;
- описание;
- центральный поиск.

3. Блок "Сегодня":
- 🕒 Время;
- 📅 Календарь;
- 🌤 Погода.

4. Блок "Основные разделы":
- 📜 Chronicles of Black Desert;
- 🎬 Entertainment;
- 🔧 Services;
- 🤖 AI.

5. О проекте.

6. Footer.
