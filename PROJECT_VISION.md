# StartWave 🌊

## Видение проекта

StartWave — это персональная стартовая площадка в интернете.

Цель проекта — создать удобное место, где человек может найти полезные сервисы, информацию, развлечения и инструменты в одном понятном интерфейсе.

StartWave не является только игровым сайтом. Это универсальная домашняя страница для пользователя.

---

# Миссия проекта

Сделать интернет удобнее:

- меньше лишнего поиска;
- больше полезной информации;
- понятная структура;
- удобный доступ к нужным инструментам.

---

# Для кого StartWave

## Обычные пользователи

Люди, которым нужна удобная стартовая страница:

- сервисы;
- развлечения;
- полезные сайты;
- инструменты.

## Игроки

Особенно игроки Black Desert Online.

Проблема:
информация разбросана по разным источникам.

Решение:
создать понятную базу знаний:

- классы;
- ресурсы;
- профессии;
- крафт;
- фарм;
- механики игры.

## Пользователи AI

Раздел с полезными AI-инструментами и возможностями.
