# StartWave 🌊

> **Твой день. Твои сервисы. Твои идеи.**

## О проекте

StartWave — современная цифровая стартовая страница и персональный хаб для информации, сервисов, развлечений, игр и AI-инструментов.

Проект помогает быстрее находить нужное и переходить от идеи к действию, не пытаясь заменить существующие сервисы.

## Возможности

- текущее время и дата;
- поиск через Яндекс;
- быстрый доступ к основным разделам;
- Entertainment;
- Services;
- AI;
- материалы по Black Desert Online;
- адаптивный интерфейс.

## Текущий статус

Проект находится в активной разработке. Основная структура и визуальный стиль StartWave 1.0 созданы; продолжаются документирование, тестирование и развитие разделов.

## Основные файлы проекта

- `index.html` — главная страница;
- `assets/css/style.css` — общие стили;
- `assets/js/script.js` — часы, дата и клиентская логика;
- `pages/` — вложенные страницы BDO;
- `assets/images/` — изображения проекта.

## Документация

- [PROJECT_SPEC.md](PROJECT_SPEC.md) — проверяемые требования StartWave 1.0;
- [PROJECT_VISION.md](PROJECT_VISION.md) — миссия и долгосрочное развитие;
- [ROADMAP.md](ROADMAP.md) — этапы, приоритеты и статусы разработки;
- [TESTING.md](TESTING.md) — результаты проверок.

## Локальный запуск

Проект состоит из статических HTML, CSS и JavaScript. Для корректной проверки рекомендуется открыть корень проекта через локальный HTTP-сервер и перейти на `index.html`.
